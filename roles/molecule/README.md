Role Name
=========

molecule

Installs molecule 9and ansible) in a python venv for the  specified user

Requirements
------------

None.

Role Variables
--------------

user_account - the user for which to install molecule

Dependencies
------------

None.

Example Playbook
----------------

    - hosts: all
      roles:
         - role: molecule
